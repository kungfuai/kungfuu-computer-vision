9.2.1. Self Supervised Learning and Linear Probing on CIFAR10#

In this notebook, we use the MOCOV3 method to train a model in a self-supervised learning (SSL) fashion, and evaluate the model by the classification accuracy of linear probing. We use the solo_learn library 1 (alternatives: VISSL, lightly) which has a good coverage of recent self-supervised learning algorithms.
